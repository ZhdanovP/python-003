Chess game, python implementation

class chess.Bishop(color)¶

Bishop class implementation. The bishop can move as many squares, in a diagonal path, as long as there is no piece blocks that path.

class chess.ChessPiece(color)¶

Superclass for all chess pieces.

class chess.King(color)¶

King class docstring. The King can move one square in any direction

class chess.Knight(color)¶

Knight class implementation. The knight can move two squares in one direction, then one square at a 90 degree angle to that second square. Unlike other pieces, the Knight can jump other pieces that are in it’s path.

class chess.Pawn(color)¶

Pawn class implementation. The pawn can move forward one square. It can only attack the square that is in the forward-diagonal position to it’s location. If a pawn encounters a piece in the square in front of it, it can no longer

move.On a pawns first move, it can move two squares forwards instead of one, as long as it does not inferior with the previous rule.

class chess.Queen(color)¶

Queen class implementation. The Queen can move as many squares, in any single direction, as long there is no piece blocks that path.

class chess.Rook(color)¶

Rook class implementation. The rook can move as many squares, forwards, backwards, or sideways, as long as there is no piece blocking that path.
